# Architecture Decision Records (ADR)

Registro de decisiones de arquitectura relevantes. Cada ADR es inmutable una vez
aprobado; si una decisión cambia, se crea un ADR nuevo que *supersede* al anterior.

Formato: basado en [Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions.html).
